To build the docs webpage locally:

```bash
pip install sphinx sphinx-rtd-theme
cd docs
make html
```

Open `build/html/index.html` in your browser.

Documentation available at: [ALATDYN](https://aladyn.readthedocs.io/en/latest/index.html)
